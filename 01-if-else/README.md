# Estruturas Condicionais - if/else

Exercícios desenvolvidos para praticar estruturas condicionais em Python.

## Conceitos praticados

- if
- elif
- else
- Operadores relacionais
- Operadores lógicos

## Exercícios

- Cálculo do triplo de um número
- Verificação de número par ou ímpar
- Verificação da situação de um aluno com base na nota

## Objetivo

Desenvolver raciocínio lógico e compreender a tomada de decisões em programas Python.
