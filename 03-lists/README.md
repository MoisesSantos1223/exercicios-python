# Listas

Esta pasta contém exercícios desenvolvidos para praticar o uso de listas em Python.

## Conceitos estudados

- Criação de listas
- Adição de elementos com `append()`
- Armazenamento de dados
- Busca de valores
- Funções `max()` e `min()`
- Soma de elementos
- Contagem de valores
- Filtragem de dados

## Objetivo

Aprender a armazenar múltiplos valores em uma única estrutura de dados e manipular essas informações de forma organizada e eficiente.

## Exercícios realizados

Os exercícios desta pasta abordam situações como:

- Armazenamento de números digitados pelo usuário
- Identificação do maior e menor valor
- Soma de elementos
- Contagem de valores
- Filtragem de números pares e ímpares
- Estatísticas básicas de uma lista
