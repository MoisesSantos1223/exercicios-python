Esta pasta contém os exercícios e exemplos que desenvolvi durante meus estudos sobre importação de módulos em Python.

Nesta etapa do curso, aprendi como utilizar módulos e bibliotecas para aproveitar recursos já disponíveis na linguagem, tornando os programas mais organizados, reutilizáveis e eficientes.

## Objetivos

- Aprender a utilizar a instrução `import`
- Entender como funcionam os módulos em Python
- Utilizar funções disponíveis em bibliotecas da linguagem
- Praticar a reutilização de código
- Compreender a importância da modularização

## Conteúdos Estudados

- Importação de módulos
- Importação de funções específicas
- Biblioteca `math`
- Utilização de apelidos com `as`
- Modularização de programas
