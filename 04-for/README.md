# Estruturas de Repetição - For

Esta pasta contém exercícios desenvolvidos para praticar o uso da estrutura de repetição `for` em Python.

## Conceitos estudados

- Laço `for`
- Função `range()`
- Contadores
- Repetições controladas
- Estruturas condicionais dentro de laços
- Processamento de sequências

## Objetivo

Aprender a utilizar o laço `for` para executar repetições de forma organizada e eficiente, compreendendo como percorrer sequências e resolver problemas que exigem múltiplas execuções de uma mesma instrução.

## Exercícios realizados

Os exercícios desta pasta abordam situações como:

- Contagem crescente e decrescente
- Tabuadas
- Soma de valores
- Contagem de números pares e ímpares
- Validação de condições
- Processamento de dados utilizando repetições

## Observação

Os exercícios presentes nesta pasta foram desenvolvidos com o objetivo de fortalecer minha lógica de programação e registrar minha evolução durante os estudos de Python.
