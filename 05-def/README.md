# Funções (Def)

Esta pasta contém exercícios desenvolvidos para praticar a criação e utilização de funções em Python.

## Conceitos estudados

- Criação de funções com `def`
- Chamada de funções
- Organização de código
- Reutilização de código
- Parâmetros
- Retorno de valores com `return`
- Separação de responsabilidades

## Objetivo

Aprender a dividir problemas em partes menores utilizando funções, tornando o código mais organizado, reutilizável e fácil de manter.

## Exercícios realizados

Os exercícios desta pasta abordam situações como:

- Exibição de mensagens
- Cálculos matemáticos
- Menus interativos
- Validação de dados
- Cadastro de informações
- Pequenos sistemas utilizando funções

## Benefícios das funções

- Evitam repetição de código
- Facilitam a manutenção
- Melhoram a organização do programa
- Tornam o código mais legível
- Permitem reutilizar soluções em diferentes partes do projeto

## Observação

Os exercícios presentes nesta pasta foram desenvolvidos com o objetivo de fortalecer minha lógica de programação, praticar a modularização de código e registrar minha evolução durante os estudos de Python.
